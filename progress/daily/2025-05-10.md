# 📅 2025-05-10 문제 풀이 로그

## ✅ 푼 문제

- [x] 1753 - 최단경로 (백준)  
  → 단일 시작점에서 모든 노드까지의 최단 경로를 구하는 문제  
  → 우선순위 큐를 활용한 다익스트라 알고리즘 사용  
  → 인접 리스트 기반 그래프 구성  
  → 시작 정점에서 각 노드까지의 거리를 저장하는 distance 배열 활용  
  → 거리 갱신 시, 현재 저장된 거리보다 작을 경우만 업데이트하여 불필요한 연산 방지  
  → 거리 배열의 값이 무한(INF)일 경우, 도달 불가로 간주하고 "INF" 출력

## 🧠 짧은 회고

- 기본 다익스트라 구현 문제로, 코딩 테스트 준비에서 필수적인 유형이었다  
- `heapq`를 이용해 효율적으로 최소 거리를 우선 탐색할 수 있는 구조가 잘 와닿았다  
- 입력 데이터가 많은 만큼 `sys.stdin.readline`을 활용한 빠른 입력 처리의 중요성을 체감했다  
- 다익스트라 알고리즘의 구조와 동작 원리를 실전 코드로 반복 연습하기에 좋은 문제였다
