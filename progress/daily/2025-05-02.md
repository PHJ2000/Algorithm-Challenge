# 📅 2025-05-02 문제 풀이 로그

## ✅ 푼 문제

- [x] 9251 - LCS (백준)  
  → 두 문자열 사이의 가장 긴 공통 부분 수열(Longest Common Subsequence)을 구하는 문제  
  → 2차원 DP를 활용해 이전 상태를 누적하며 최적 해를 구함  
  → 같은 문자가 등장하면 이전 대각선에서 +1, 다르면 왼쪽/위쪽 중 최댓값을 선택

- 작은 문제부터 누적해가는 방식으로 구조화된 사고를 연습할 수 있었던 좋은 문제였다.
